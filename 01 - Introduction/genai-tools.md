# AI Tools Overview: ChatGPT and Other Generative AI Tools

This section provides an overview of some popular AI tools that demonstrate the practical applications of generative AI. We will focus primarily on **ChatGPT**, along with two other useful tools: **DeepAI** and **Descript**.

---

# ChatGPT

ChatGPT is one of the most widely used generative AI tools. It allows users to interact with an AI model by entering prompts and receiving responses in natural language.

For example, a user could enter the following prompt:

> Write a small poem mimicking the style of Shakespeare.

ChatGPT will then generate a response based on the request.

---

# ChatGPT Interface Overview

## Chat History

One of the useful features in ChatGPT is the **chat history panel**.

This panel allows users to:
- View previous conversations
- Reopen earlier chats
- Continue discussions from earlier sessions

This makes it easy to organize and revisit past work.

---

# ChatGPT Settings

ChatGPT provides several settings that allow users to customize their experience.

These include:

### Theme Settings

Users can change the visual theme of the interface.

Available options typically include:
- Light Mode
- Dark Mode

### Language Settings

Users can select their preferred language for interaction.

### Personalization

ChatGPT allows personalization options so the system can adapt to user preferences.

### Voice Responses

Some versions allow responses in **audio format**, enabling ChatGPT to speak responses instead of only displaying text.

Users can also choose between different voice styles.

---

# Data Controls and Privacy

ChatGPT includes an important setting called:

**Improve the model for everyone**

When this option is enabled, the system may use user conversations to help improve the model.

This means:
- Prompts
- Conversations
- User inputs

may be analyzed to enhance the AI.

However, this raises potential **privacy considerations**.

If users prefer stronger privacy protection, they can disable this option. When disabled, conversations are not used to improve the model.

---

# Connecting External Applications

ChatGPT can integrate with external storage services such as:

- Google Drive
- Dropbox

This allows the system to access files, documents, or images stored in those services.

Such integrations make it easier to work with external data.

---

# Security Features

ChatGPT also provides security options including **Multi-Factor Authentication (MFA)**.

When MFA is enabled, logging in may require additional verification such as:

- A confirmation code sent via SMS
- A mobile authentication prompt

This adds an extra layer of protection to the user account.

---

# Exporting Data

ChatGPT allows users to **export their data**.

Exported data may include:
- Conversation history
- AI-generated responses
- Associated files

This feature can be useful for:

- Creating backups
- Saving important outputs
- Archiving work

However, users should remember that exported files will be stored locally on their device and may contain sensitive information.

---

# ChatGPT Free vs ChatGPT Plus

There are two main versions of ChatGPT available.

## Free Version

The default version of ChatGPT is free.

Features include:
- Text generation
- Question answering
- Translation
- Code generation
- General conversation

However, the free version may include:
- Usage limits
- Slower responses during high traffic
- Access to a baseline model

---

## ChatGPT Plus

ChatGPT Plus is a paid subscription.

Cost:
- **$20 per month**

Benefits include:
- Priority access during peak usage
- Faster response times
- Higher usage limits
- Access to more advanced models

The free version often uses **GPT-3.5**, while ChatGPT Plus provides access to **GPT-4**, which offers stronger reasoning and a larger context window.

For most learning purposes, the **free version is sufficient**.

---

# DeepAI – Image Generation Tool

Another interesting AI tool is **DeepAI**, which allows users to generate images using text prompts.

It is a free tool, although users may encounter advertisements or pop-ups.

To generate an image, users simply enter a prompt describing what they want to create.

Example prompt:

> A futuristic city at sunset with flying cars.

The system will generate an image based on the description.

---

## Speed vs Quality

DeepAI allows users to choose between:

- **Speed** – faster image generation
- **Quality** – higher quality images

The quality of the generated image depends heavily on the prompt.

More specific prompts typically produce better results.

---

# Descript – AI Audio and Video Editing Tool

**Descript** is another AI-powered tool that focuses on audio and video editing.

It allows users to edit media content without requiring advanced video editing skills.

Descript can be used for:

- Video editing
- Podcast production
- Audio editing
- Screen recording

---

# Installing Descript

Descript must be **downloaded and installed locally** on your computer.

It offers a **free version**, although some features may be limited.

Once installed, users can create projects such as:

- Video projects
- Audio recordings
- Podcast episodes

---

# Example Feature: Background Removal

One useful feature in Descript is the **Green Screen effect**.

Example workflow:

1. Upload a video.
2. Open the **Effects** panel.
3. Enable **Green Screen**.

Descript will automatically remove the background from the video.

If the video was recorded using an actual green screen, the results will be even better.

---

# Additional Features

Descript also includes several AI-powered tools such as:

- Eye contact correction (adjusts the speaker’s gaze to appear directed at the camera)
- Automatic transcription
- Audio cleanup
- Editing video by editing text

These features make video production easier even for beginners.
